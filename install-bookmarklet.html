Kindle Highlights Exporter

Drag this button to your bookmarks bar:

Export Kindle Highlights

How to use

  1. Drag the orange button above to your bookmarks bar
  2. Go to read.amazon.com/notebook and log in
  3. Click the "Export Kindle Highlights" bookmark
  4. Wait while it scrapes all your books (progress shown in top-right corner)
  5. A kindle-export.json file will download automatically
  6. Move the JSON to kindle-highlights/raw/ and run: node organize.js
Tip: The scraper takes about 1.5 seconds per book. If you have 200+ books, expect it to take 5-6 minutes.
